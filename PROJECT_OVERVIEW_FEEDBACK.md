Kami membangun sistem ini dengan pendekatan **Microservices** agar tangguh, modular, dan ringan.

### 1. The Gateway: WhatsApp Interaction (WAHA)

Pintu masuk utama adalah **WhatsApp**. Kami tidak memaksa pelanggan install aplikasi baru.

- **Integrasi**: Menggunakan _WAHA (WhatsApp HTTP API)_.
- **Fitur Unik**: **Webhook Router**.
  - Sistem bisa membedakan chat "Curhat" vs "Orderan".
  - Otomatis mendeteksi lampiran file dokumen.

### 2. The Brain: Backend Dispatcher (Node.js)

Server pusat yang tidak pernah tidur.

- Mengatur antrian (Queue) menggunakan **Redis** agar server tidak meledak saat banjir order.
- Mengarahkan tugas ke departemen yang benar (Worker).

### 3. The Hands: Processing Engine (Python)

Ini adalah fitur unggulan kami. Menggunakan **Deterministic AI** (Rule-Based), bukan LLM yang suka halusinasi.

- **Template Scanner**: Sistem bisa "belajar" dari file contoh (Golden Sample) yang diupload pemilik toko.
- **Two-Input Applicator**: Jika pelanggan kirim file rusak, sistem mengambil "DNA Format" dari file contoh, lalu menyuntikkannya ke file pelanggan.
  - _Result_: File pelanggan otomatis rapi (Margin 4433, Font Times New Roman) dalam hitungan detik.
